ASSIGNMENT 03 Requirements

This game is essentially the 2D gameplay of the character from the first person shooter, Overwatch. "PHarah" is a character with a jetpack, and shoots rockets. I implemented a basic jetpack flying system, with gravity and acceleration. I also implemented the fuel meter, which can go below zero and leave the player stranded without fuel to be hit by the enemies.

  1. Theme - Pharah, Overwatch
  2. User controlled character - A / D, spacebar for fuel, R for reload
  3. Objective - shoot all enemies
  4. Obstacle - enemies on the ground
  5. Scoring system - counts how many enemies shot, top left corner
  6. HTML Interface - below
  7. Start Screen / Game Screen / Gameover Screen
  8. Sound file - every shot (click) plays a sound.
  9. External graphics - Assets of Overwatch, Destiny, Google Images
